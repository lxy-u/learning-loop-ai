# Figma 原型设计指南

本指南说明如何为 Learning Loop AI 项目创建丰富的 Figma 原型设计。

## 设计目标

- **展示产品愿景**: 完整的功能规划和用户体验
- **体现设计能力**: 2026年顶级设计标准
- **指导开发方向**: 为后续功能提供设计参考
- **吸引潜在用户**: 通过精美设计展示产品价值

## 设计系统

### 颜色系统

#### 主色调
```
- 背景: #0A0A0A (深邃黑)
- 表面: #1A1A1A (深灰)
- 表面浅: #2D2D2D (中灰)
- 主色: #6366F1 (靛蓝色)
- 主色浅: #818CF8 (浅靛蓝)
```

#### 功能色
```
- 成功: #10B981 (祖母绿)
- 错误: #EF4444 (陶土红)
- 警告: #F59E0B (琥珀色)
- 信息: #3B82F6 (蓝色)
```

#### 文字色
```
- 主文字: #F9FAFB (白色)
- 次要文字: #9CA3AF (浅灰)
- 禁用文字: #6B7280 (中灰)
```

### 字体系统
```
- 标题: Inter, 600-700, 24-36px
- 正文: Inter, 400-500, 16px
- 说明: Inter, 400, 14px
- 代码: JetBrains Mono, 400, 14px
```

### 间距系统
```
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- xxl: 48px
```

### 圆角系统
```
- 小: 6px
- 中: 8px
- 大: 12px
- 特大: 16px
- 全圆: 9999px
```

## 页面设计

### 1. 首页

#### 布局结构
```
┌─────────────────────────────────────────┐
│  导航栏 (Logo + 用户菜单)              │
├─────────────────────────────────────────┤
│  Hero 区域 (标题 + 描述 + CTA)        │
├─────────────────────────────────────────┤
│  功能展示 (3-4 个核心功能卡片)      │
├─────────────────────────────────────────┤
│  使用流程 (步骤图示)                │
├─────────────────────────────────────────┤
│  技术栈展示 (图标 + 说明)            │
├─────────────────────────────────────────┤
│  底部信息 (版权 + 社交链接)        │
└─────────────────────────────────────────┘
```

#### 核心组件
- **导航栏**:
  - Logo (Learning Loop AI)
  - 首页链接
  - 功能介绍链接
  - 用户菜单 (登录/注册)

- **Hero 区域**:
  - 主标题: "AI驱动的学习闭环系统"
  - 副标题: "从听懂到会做，一次完成"
  - CTA 按钮: "开始学习"
  - 背景动画: 粒子效果

- **功能卡片**:
  - 智能讲解
  - 题目生成
  - 错题强化
  - 学习统计

### 2. 学习页

#### 布局结构
```
┌─────────────────────────────────────────┐
│  顶部导航 (返回 + 标题)            │
├─────────────────────────────────────────┤
│  输入区域 (大输入框 + 生成按钮)    │
├─────────────────────────────────────────┤
│  知识点讲解 (卡片展示)             │
├─────────────────────────────────────────┤
│  题目区域 (题目 + 选项)            │
├─────────────────────────────────────────┤
│  反馈区域 (结果 + 分析)              │
├─────────────────────────────────────────┤
│  操作按钮 (再来一道 + 保存笔记)        │
└─────────────────────────────────────────┘
```

#### 交互状态
- **初始状态**: 显示输入区域
- **加载状态**: 显示加载动画
- **讲解状态**: 显示知识点内容
- **答题状态**: 显示题目和选项
- **反馈状态**: 显示结果和分析
- **强化状态**: 显示新题目


## 组件库

### 基础组件

#### 按钮
- **主要按钮**: 靛蓝色背景，白色文字
- **次要按钮**: 透明背景，靛蓝色边框
- **危险按钮**: 红色背景，白色文字
- **图标按钮**: 圆形，图标居中

#### 输入框
- **文本输入**: 大尺寸，圆角，聚焦效果
- **选择框**: 下拉样式，自定义箭头
- **搜索框**: 搜索图标，实时过滤

#### 卡片
- **信息卡片**: 深灰背景，圆角，阴影
- **功能卡片**: 悬停效果，图标装饰
- **统计卡片**: 渐变背景，数据突出

#### 弹窗
- **确认弹窗**: 标题 + 内容 + 操作按钮
- **输入弹窗**: 表单样式，验证提示
- **加载弹窗**: 动画效果，进度指示

### 复杂组件

#### 学习进度条
- **总进度**: 显示整体学习进度
- **单题进度**: 显示当前题目进度
- **动画效果**: 平滑过渡，颜色变化

#### 知识点标签
- **标签样式**: 圆角，彩色背景
- **点击交互**: 筛选对应题目
- **数量徽章**: 显示该标签下的题目数

#### 用户头像
- **默认头像**: 字母缩写，随机颜色
- **自定义头像**: 上传功能，裁剪预览
- **等级标识**: 勋章样式，动态颜色

## 交互设计

### 动画效果

#### 页面过渡
- **淡入淡出**: 页面切换时使用
- **滑入滑出**: 侧边栏展开/收起
- **缩放效果**: 卡片点击反馈

#### 元素动画
- **悬停效果**: 卡片上浮，阴影加深
- **点击效果**: 按钮按下，颜色加深
- **加载动画**: 旋转圆环，脉冲效果

### 响应式设计

#### 断点设置
```
- 手机: < 640px
- 平板: 640px - 1024px
- 桌面: > 1024px
```

#### 适配策略
- **手机端**: 单列布局，简化导航
- **平板端**: 双列布局，侧边栏可收起
- **桌面端**: 三列布局，完整功能展示

### 无障碍设计
- **键盘导航**: Tab 键顺序合理
- **屏幕阅读器**: 语义化标签，ARIA 属性
- **色彩对比**: 符合 WCAG AA 标准
- **字体大小**: 支持系统缩放

## 设计验证

### 用户测试
- **可用性测试**: 5-10 人测试
- **A/B 测试**: 不同设计版本对比
- **反馈收集**: 收集用户意见

### 技术验证
- **性能测试**: 加载速度，动画流畅度
- **兼容性测试**: 不同浏览器和设备
- **可访问性测试**: 屏幕阅读器，键盘导航

## 设计亮点

### 视觉亮点
- **2026年设计标准**: 深色主题，玻璃态效果
- **动态背景**: 粒子动画，营造沉浸感
- **精致细节**: 阴影层次，圆角统一

### 交互亮点
- **流畅动画**: 所有交互都有平滑过渡
- **即时反馈**: 用户操作立即得到响应
- **智能引导**: 新用户引导，功能提示

### 功能亮点
- **完整闭环**: 讲解 → 出题 → 强化 → 统计
- **个性化**: 根据用户学习数据推荐内容
- **社交化**: 学习成就分享，好友PK
