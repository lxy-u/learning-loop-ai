这个项目的想法很简单：通过'讲解 → 出题 → 错题强化'的闭环，帮助备考用户从'听懂'到'会做'。

我用了 3 天时间，从 0 到 1 完成了这个项目，今天我想把这个过程分享给大家。"

### 项目介绍 (0:30-1:30)

**画面**: 项目演示 + 功能展示

**旁白**:
"Learning Loop AI 的核心功能有三个：

第一，智能知识点讲解。用户输入任意知识点，AI 会生成结构化讲解，包括通俗解释、举例说明、常见误区。

第二，智能题目生成。基于同一知识点生成选择题，4 个选项，包含正确答案，即时答题反馈。

第三，错题强化学习。错误时展示错因分析，提供正确思路指导，'再来一道'功能生成相似新题。

整个学习流程非常流畅，用户只需要输入知识点，就能完成从学习到练习的完整闭环。"

**画面**: 功能演示动画 + 用户流程图

### 技术栈介绍 (1:30-2:30)

**画面**: 技术栈展示 + 架构图

**旁白**:
"这个项目使用的技术栈非常现代化：

前端使用 Next.js 16.2.1 和 React 19.2.4，这是最新的版本，性能和开发体验都非常好。

样式方案使用原生 CSS + 设计系统，没有使用任何 UI 库，这样可以完全控制设计。

后端使用 Next.js API Routes，非常简洁。

数据库使用 Supabase，这是一个基于 PostgreSQL 的 BaaS 平台，非常适合个人项目。

部署到 Vercel，一键部署，自动绑定域名。

数据分析使用 Plausible，这是一个隐私友好的分析工具。

用户反馈使用 Tally，这是一个现代的表单工具。

整个技术栈非常现代化，开发体验非常好。"

**画面**: 技术栈动画 + 构构图
