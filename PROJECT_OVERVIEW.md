# Learning Loop AI 项目全面介绍

## 1. 整体架构

### 1.1 技术架构

**前端技术栈**
- Next.js 16.2.1 (App Router 架构)
- React 19.2.4
- TypeScript
- CSS-in-JS 内联样式
- 响应式设计

**文件结构**
```
app/
├── prototype/
│   ├── components/          # 通用组件
│   ├── auth/              # 认证页面
│   ├── course/            # 课程页面
│   ├── quiz/              # 练习页面
│   ├── exam/              # 考试页面
│   ├── wrong-questions/   # 错题本页面
│   ├── graph/             # 知识图谱页面
│   ├── membership/        # 会员中心
│   ├── community/         # 社区页面
│   ├── profile/           # 个人中心
│   ├── settings/          # 设置页面
│   ├── path/              # 学习路径页面
│   ├── report/            # 学习报告页面
│   ├── leaderboard/       # 排行榜页面
│   └── page.tsx           # 仪表盘
├── layout.tsx             # 全局布局
└── page.tsx               # 主页
```

### 1.2 系统架构

**核心模块**
1. **用户认证系统**：支持注册、登录、忘记密码
2. **学习管理系统**：课程学习、练习、考试、错题管理
3. **智能推荐系统**：基于AI的学习路径推荐
4. **会员管理系统**：多级会员体系和权益管理
5. **社区互动系统**：学习社区、排行榜
6. **数据分析系统**：学习数据统计和可视化

**数据流**
- 前端：React 组件 + 状态管理 (useState, useEffect)
- 后端：待实现（规划中使用 Supabase）
- 数据存储：待实现（规划中使用 Supabase 数据库）

## 3. 付费机制

### 3.1 会员体系

| 版本 | 价格 | 权益 |
|------|------|------|
| **免费版** | ¥0/永久 | 基础学习功能、100道练习题、基础学习报告 |
| **高级版** | ¥99/月<br>¥899/年 (省30%) | 全部基础功能、无限练习题、高级学习报告、高级AI推荐、专业学习路径、优先客服支持 |
| **专业版** | ¥199/月<br>¥1799/年 (省30%) | 全部高级功能、1对1专家辅导、定制学习计划、模拟考试批改、考试报名指导、专属学习群 |

### 3.2 付费流程
1. 用户进入会员中心
2. 选择月付或年付模式
3. 选择会员套餐
4. 完成支付
5. 系统自动开通会员权益

### 3.3 退款政策
- 订阅后7天内可申请全额退款
- 超过7天后不予退款
- 取消订阅后，当前会员权益持续到订阅周期结束

## 7. 设计理念

### 7.1 视觉设计
- **色彩系统**：主色调 #3B82F6（蓝色），背景 #F5F7FA，强调色 #1E40AF
- **排版系统**：清晰的字体层次，标题使用较大字号和粗体
- **布局系统**：卡片式布局，合理的留白和间距
- **图标系统**：使用直观的 emoji 和图标增强视觉表达

### 7.2 响应式设计
- 桌面端：多列布局，充分利用屏幕空间
- 平板端：自适应布局，保持核心功能可见
- 移动端：单列布局，优化触摸交互

### 7.3 无障碍设计
- 键盘导航支持
- 屏幕阅读器兼容
- 色彩对比度优化
- 文字大小调整选项

### 7.4 品牌一致性
- 统一的视觉风格
- 一致的交互模式
- 专业的教育品牌形象
- 国际化设计语言

## 总结

Learning Loop AI 是一个面向专业考试备考的智能学习平台，通过结合现代前端技术和AI驱动的个性化推荐，为用户提供高效、专业的学习体验。平台采用国际化设计语言，注重用户体验和功能创新，构建了完整的学习生态系统。

核心优势在于：
- 个性化学习路径推荐
- 智能错题分析
- 知识图谱可视化
- 多级会员体系
- 活跃的学习社区

未来，Learning Loop AI 将继续通过技术创新和内容扩展，成为专业考试备考领域的领先平台，为更多考生提供优质的学习解决方案。
